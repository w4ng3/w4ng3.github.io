Express 是一个最小且灵活的 Node.js Web 应用程序框架，它为 Web 和移动应用程序提供了一组强大的功能。
使用 Express 可以快速地搭建一个完整功能的网站。

Express 框架核心特性：
- 可以设置中间件来响应 HTTP 请求。
- 定义了路由表用于执行不同的 HTTP 请求动作。
- 可以通过向模板传递参数来动态渲染 HTML 页面。
:::
